What is this?

The Personal AI Playbook is a self-contained system for reusing AI prompts instead of reinventing them every time. It runs entirely in this browser file — no account, no server, no API calls. Everything you save stays on this device via local storage.

  • Dashboard — your interview answers and a shortlist of workflows picked for you.
  • Workflow Library — a built-in set of prompt templates, grouped by task. Copy one, fill in the [brackets], done.
  • Prompt Builder — build a prompt from labeled parts (role, objective, constraints…) with a live preview of the assembled text.
  • Loop Builder — turn any prompt into one that re-checks and improves its own output automatically, with rules for when to stop.
  • My Workflows — anything you save, create, or duplicate, searchable and favoritable.

Use / to jump to search, N for a new workflow, Esc to close dialogs.
